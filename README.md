# PacificWaves

An interactive North Pacific swell simulation for O'ahu's North Shore.

## Overview

PacificWaves is a web-based swell simulation tool that models wave propagation across the North Pacific Ocean to O'ahu's North Shore. The simulator allows users to place storm systems and observe how swell energy travels across the ocean, providing insights into surf forecasting dynamics.

## Features

- Interactive canvas for placing storm systems
- Real-time swell propagation visualization
- Configurable storm parameters (intensity, duration, direction)
- Visual representation of swell energy reaching O'ahu's North Shore
- Built entirely with HTML5, CSS3, and vanilla JavaScript

## Usage

Open `waves.html` in a modern web browser to run the simulation.

## Technology

- Pure HTML5 Canvas for rendering
- Vanilla JavaScript (no framework dependencies)
- CSS Grid and Flexbox for responsive layout

## About

This tool is designed for surfers, forecasters, and anyone interested in understanding how North Pacific swells develop and travel to Hawaiian shores.
